5. COMMUNITY PULSE & RISKS (The "Don't Do It" List)

Estado Tema Detalle Técnico + Insight Consultor
🔴 Jira Software: BUG: En la versión cloud, usuarios están reportando que las transiciones de estado de los issues a veces fallan sin un error visible, requiriendo un refresco de la página y un reintento. Esto ocurre esporádicamente pero es molesto.
Insight Consultor: Un bug intermitente en una funcionalidad central como las transiciones puede causar frustración en los usuarios y pérdida de confianza en la plataforma. Recomendación: Monitorear activamente las discusiones en la comunidad.atlassian.com. Aconsejar a los clientes que registren estos incidentes con capturas de pantalla o videos para Atlassian. Para el consultor, es un riesgo a mencionar en implementaciones grandes, asegurando que los equipos estén al tanto y documenten cualquier ocurrencia.
🔴 API de Confluence Cloud (v1) Deprecación: Atlassian ha anunciado la deprecación formal de la API REST de Confluence Cloud V1 para el 30 de Septiembre de 2026. Todas las aplicaciones y scripts deben migrar a la API V2.
Insight Consultor: Esta es una alerta crítica para arquitectos y desarrolladores. Si hay integraciones personalizadas o aplicaciones de terceros que usan la API V1, estas se romperán. Oportunidad: Ofrecer servicios de auditoría de integraciones y migración de APIs para clientes. Riesgo: Ignorar esto causará interrupciones significativas en el negocio para cualquier proceso que dependa de la API V1. Planificar el cambio ahora es esencial.
🟡 Discusiones (Precios) Un hilo popular en la comunidad discute el impacto de las nuevas licencias "Premium" para Atlassian Rovo y Atlassian Focus. Los usuarios expresan preocupación por el aumento de los costes para acceder a las funcionalidades de IA, sugiriendo que las características básicas de IA deberían ser parte de los planes estándar.
Insight Consultor: Esto indica una posible resistencia al modelo de precios de Atlassian para las nuevas funcionalidades de IA. Consejo: Al presentar Rovo o Focus, es crucial justificar claramente el valor añadido de las características Premium en relación con su coste. Anticipar estas objeciones y preparar argumentos sólidos sobre el ROI para planes empresariales. Es una alerta sobre la sensibilidad al precio en la adopción de nuevas tecnologías.
🟡 Discusiones (IA Ética) Aumentan las discusiones sobre la ética y la privacidad de los datos utilizados por las nuevas características de IA en Jira, Confluence y Rovo, especialmente en relación con la gobernanza de datos en entornos altamente regulados.
Insight Consultor: Este es un riesgo creciente, especialmente para empresas en sectores con estrictas normativas de privacidad (ej. salud, finanzas). Acción: Como consultor, debe estar preparado para explicar la postura de Atlassian sobre la privacidad y la seguridad de los datos en relación con la IA. Asegurarse de que los clientes comprenden cómo se utilizan sus datos y qué controles tienen. Podría ser un freno en la adopción si no se abordan las preocupaciones de manera proactiva y transparente.
