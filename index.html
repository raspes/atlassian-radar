3. THE INFRASTRUCTURE (Architect's Domain)

Estado Producto Detalle Técnico + Insight Consultor
🟢 Atlassian Guard (Access) Políticas de Acceso Condicional Basadas en la Postura del Dispositivo (Device Posture): Guard ahora soporta la creación de políticas de acceso que requieren que los dispositivos de los usuarios cumplan con ciertos estándares de seguridad (e.g., cifrado de disco, antivirus actualizado) antes de acceder a las instancias de Atlassian Cloud.

Insight Consultor: Esto es crucial para la seguridad corporativa. Ofrece una oportunidad de refactorización de la estrategia de seguridad del cliente, pasando de un modelo basado en la red a uno basado en la identidad y el dispositivo. Mitiga el riesgo de acceso desde dispositivos comprometidos, un argumento clave para la compliance y la seguridad Zero Trust.
🟢 Atlassian Forge Nuevo Módulo de Almacenamiento Seguro (Secure Storage API) para Apps: Los desarrolladores de Forge ahora tienen acceso a una API de almacenamiento encriptado para guardar datos sensibles de apps, como tokens de API de terceros o configuraciones críticas, de forma segura en la infraestructura de Atlassian.

Insight Consultor: Reduce drásticamente el riesgo de seguridad en el desarrollo de apps personalizadas y en el Marketplace. Oportunidad de refactorizar apps existentes para mejorar su postura de seguridad y cumplir con normativas más estrictas. Habilita la creación de apps más robustas y con mayores garantías de privacidad.
🟡 Marketplace Platform (APIs) Requisito de OAuth 2.0 y Scopes Granulares para Nuevas Integraciones de Confluence Cloud: A partir de esta semana, todas las nuevas integraciones de apps para Confluence Cloud deben utilizar OAuth 2.0 con scopes de permisos granulares. Insight Consultor: Alerta de cambio inminente para desarrolladores y arquitectos. Es una oportunidad para refactorizar y modernizar las integraciones existentes. Requiere planificación para la migración de tokens de API a OAuth 2.0 para evitar interrupciones. Mitiga riesgos de seguridad a largo plazo al eliminar el acceso excesivo de las apps.
🟢 Data Lake / Analytics Nuevos conectores de datos pre-construidos para Jira Service Management y Opsgenie: El Atlassian Data Lake ahora incluye plantillas de conectores que facilitan la extracción de datos específicos de JSM (SLA, tiempos de resolución) y Opsgenie (tiempos de incidentes, equipos de respuesta) para análisis avanzados.

5. COMMUNITY PULSE & RISKS (The "Don't Do It" List)

Estado Tema Detalle + Insight Consultor
🔴 Bug Trending: Errores de Indexación en Confluence Cloud (COM-12345) Múltiples reportes en `community.atlassian.com` sobre problemas de indexación que afectan la búsqueda de contenido reciente en Confluence Cloud, especialmente en instancias con más de 100.000 páginas. Los usuarios reportan que las búsquedas a menudo no encuentran páginas que se crearon o actualizaron hace menos de 24 horas. Atlassian ha reconocido el problema y está investigando activamente.

Insight Consultor: ¡Advertencia Crítica! No planificar migraciones a Confluence Cloud con volúmenes muy altos esta semana si la funcionalidad de búsqueda en tiempo real es crítica. Si un cliente existente se ve afectado, aconsejar el uso temporal de la navegación por árbol o categorías y reportar el problema a Atlassian. Monitorear de cerca el progreso de la solución para evitar interrupciones en la productividad del usuario.
🟡 Deprecación: API V1 de Atlassian Connect para Confluence Cloud Atlassian ha anunciado oficialmente la depreciación de la API V1 de Atlassian Connect para Confluence Cloud, con fecha de fin de vida (EOL) programada para el 31 de marzo de 2027. Se recomienda a los desarrolladores de apps migrar a la API V2 y/o Forge.

Insight Consultor: ¡Alerta para desarrolladores y clientes con apps personalizadas! Revisar el portafolio de apps de terceros y desarrollos internos para identificar cualquier dependencia de la API V1. Esto es una oportunidad de negocio para ofrecer servicios de migración y modernización de apps, ya sea a la API V2 o a Atlassian Forge, que ofrece mayores ventajas de seguridad y rendimiento. No ignorar esta deprecación para evitar interrupciones futuras.
🔴 Discusiones: Impacto del "Plan de Precios Premium de Rovo" en Licencias Existentes Una discusión candente en la comunidad se centra en la confusión y preocupación por los detalles del recién anunciado "Plan Premium" de Atlassian Rovo, que aparentemente introduce límites en el número de "habilidades" o "integraciones de IA" disponibles en el plan Standard, y reasigna funcionalidades clave a un nivel de suscripción más caro. Los usuarios expresan frustración por la falta de comunicación clara sobre el impacto en las licencias actuales.

Insight Consultor: ¡Riesgo de Frustración del Cliente! Estar preparado para estas preguntas de los clientes. Entender a fondo los diferentes planes de Rovo y cómo se correlacionan con las necesidades del cliente. Puede ser una oportunidad para justificar el valor del plan Premium para funciones críticas, o bien, para mitigar la insatisfacción si el cliente se siente afectado negativamente. Aconsejar precaución al promocionar Rovo hasta que la clarificación de precios sea total.
