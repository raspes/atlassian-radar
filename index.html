3. THE INFRASTRUCTURE (Architect's Domain)

Estado Componente Detalle Técnico + Insight Consultor
🟢 Atlassian Guard (Access)

Feature: Soporte para FIDO2/WebAuthn como método de autenticación sin contraseña (passkeys) para todos los productos de Atlassian Cloud. Mejora significativamente la seguridad y la experiencia del usuario.

Insight Consultor: Oportunidad clave para modernizar la seguridad de la identidad. Permite vender una solución de IAM de última generación, eliminando las contraseñas y mitigando los riesgos asociados a credenciales débiles o robadas. Es una oportunidad de refactorización de políticas de seguridad.
🟢 Atlassian Forge

Feature: Nueva API para la gestión programática de "Custom Fields" en Jira Cloud. Los desarrolladores ahora pueden crear, actualizar y eliminar campos personalizados de forma automatizada, permitiendo una mayor flexibilidad en la integración de datos externos.

Insight Consultor: Esto desbloquea un nuevo nivel de automatización e integración para las aplicaciones de Forge. Oportunidad de refactorizar soluciones complejas que antes requerían trabajo manual o APIs menos robustas. Muy relevante para la gestión de datos.
🟡 Marketplace Platform (APIs)

Alerta/Cambio: Deprecación de la API v2 de "User Management" para aplicaciones de la Marketplace. Los desarrolladores tienen hasta el 30 de septiembre de 2026 para migrar a la API v3, que ofrece controles de privacidad más granulares.

Insight Consultor: Breaking Change Potencial. Hay que auditar de inmediato las aplicaciones de terceros o personalizadas que utilicen esta API. Oportunidad para contactar a los clientes y ofrecer servicios de consultoría para la migración y asegurar la continuidad operativa y el cumplimiento.
🟢 Data Lake / Analytics

Feature: Mejoras en el conector bidireccional entre Atlassian Data Lake y Google BigQuery. Ahora soporta replicación de datos en tiempo casi real y una mayor variedad de esquemas para análisis avanzados.

Insight Consultor: Clave para la estrategia de datos empresariales. Permite vender soluciones de Business Intelligence y Data Science sobre los datos de Atlassian, creando un valor analítico significativo. Oportunidad de integrar Atlassian en arquitecturas de datos empresariales más amplias.
🟢 Cloud Migrations (CCMA/JCMA)

Feature: El Cloud Migration Assistant (CMA) ahora soporta la migración de workflows complejos con post-functions personalizadas desde Jira Server/DC.

Insight Consultor: Reduce una barrera significativa para la migración a la nube para clientes con Jira Server/DC fuertemente personalizados. Permite vender proyectos de migración a la nube con menos riesgos y una transición más suave, especialmente para entornos maduros.
🟢 Data Residency

Feature: Anuncio de la disponibilidad de "Data Residency" para las regiones de India (Mumbai) y Brasil (São Paulo).

Insight Consultor: Oportunidad de negocio inmediata para clientes en estas regiones o con requisitos de cumplimiento específicos para estos territorios. Permite vender Atlassian Cloud a empresas que antes no podían adoptarlo por razones de soberanía de datos, abriendo nuevos mercados.
🟢 Automation

Feature: Nuevas acciones de automatización globales para interactuar con herramientas externas vía webhooks autenticados de forma nativa. Permite una orquestación cross-producto y cross-plataforma más sencilla y segura.

Insight Consultor: Permite vender una automatización empresarial más allá del ecosistema Atlassian. Oportunidad para diseñar integraciones complejas y flujos de trabajo automatizados que conecten Atlassian con sistemas de negocio externos (CRM, ERP, etc.), maximizando el valor de la plataforma.

5. COMMUNITY PULSE & RISKS (The "Don't Do It" List)

Estado Categoría Detalle Técnico + Insight Consultor
🔴 Bugs Trending

Bug Crítico: Múltiples reportes en la comunidad sobre la pérdida intermitente de "watcher" en incidencias de Jira Cloud, especialmente después de ciertas acciones masivas o automatizaciones.

Insight Consultor: Riesgo Operacional. Aconseja a los clientes revisar sus logs de auditoría y notificaciones. Si un cliente confía en los watchers para la comunicación crítica, este bug puede causar fallos de comunicación. Recomendar cautela con automatizaciones de watchers hasta que se resuelva.
🔴 Deprecaciones

Deprecación Oficial: Atlassian ha anunciado la descontinuación del soporte para "Jira Server" como plataforma en la "Política de Soporte Extendido" a partir de finales de 2026. Se insta a todos los usuarios restantes a migrar a Data Center o Cloud.

Insight Consultor: Alerta Crítica y Oportunidad Masiva. Aquellos clientes que aún operen con Jira Server deben ser contactados inmediatamente. Esta es una ventana de oportunidad final para iniciar proyectos de migración a Data Center o Cloud. El riesgo de quedarse sin soporte de seguridad es inminente.
🟡 Discusiones Calientes

Discusión: Aumento de hilos sobre el rendimiento percibido de Confluence Cloud para instancias grandes (más de 5000 usuarios) con gran cantidad de adjuntos multimedia. Usuarios reportan lentitud al cargar páginas complejas.

Insight Consultor: Alerta para clientes Enterprise de Confluence. Prepararse para abordar preguntas sobre rendimiento. Investigar las mejores prácticas de optimización de Confluence (compresión de imágenes, paginación, etc.) y las herramientas de Atlassian para monitorear el rendimiento. Podría ser un factor decisivo en grandes implementaciones.
